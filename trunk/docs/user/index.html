ClearQuest/XML Interface User Guide

Introduction

The ClearQuest XML (CQ/XML) Interface provides an interface to ClearQuest that allows you to both read and write to the ClearQuest database. The CQ/XML Interface sits on top of the ClearQuest application layer so ClearQuest business rules are still enforced.

CQ/XML communicates via XML over sockets. If you don't know either of those technologies, you're in luck. A good XML introduction is available via O'Reilly's XML.com web site or Wikipedia and we have written sample scripts utilizing sockets in both Perl and PHP.
