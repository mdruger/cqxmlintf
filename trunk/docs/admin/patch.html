  1. From ClearCase, copy the new files to an empty directory that is accesible from the CQ/XML server.
    • I always create a subdirectory in //indshare/cqxmlintf/release
    • Henceforth, this directory will be referred to as '[srcdir]'.
  2. Login to the server with root privileges.
  3. Run /opt/cqxmlintf/cprls2svr.pl -d [srcdir]
    • This script will copy the files and stop the currently running CQ/XML Interface process.
    • The CQ/XML Interface will restart within 5 minutes.
